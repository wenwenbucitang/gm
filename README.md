# gm
游戏

## 启动

在项目根目录执行：

```bash
python3 -m http.server 8080 --directory cyberpunk-megapolis
```

然后访问：<http://localhost:8080>

## 能量刀战斗

- `F`：挥砍能量刀。
- 鼠标成功锁定后，也可使用鼠标左键攻击。
- 玩家落地后会自动部署三台训练无人机；每次命中造成 40 点伤害，击毁后 4 秒重生。

## 能量刀技能

- `Q`：瞬步斩，沿瞄准方向高速突进并造成 60 点扫掠伤害。
- `C`：回旋斩，对近身范围内目标造成 45 点伤害。
- `X`：空中坠击，仅可在空中发动，落地冲击波造成 70 点伤害。
- `V`：能量波，发射可沿轨迹命中的远程刀波，造成 50 点伤害。
- 右下角技能 HUD 会显示各技能的实时冷却状态。
